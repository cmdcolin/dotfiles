# my dotfiles

A list of my dotfiles

Uses https://github.com/deadc0de6/dotdrop for managing dotfiles

Includes

- Rprofile
- ansiweather
- zshrc with zplug and dracula/zsh theme
- keychain
- vim with vim-plug

## Install

First install zplug with linuxbrew or homebrew

    brew install zplug

Use python 3

    pip install dotdrop
    dotdrop install
